# Road to Glory — Game Design Document
**버전:** 0.1 (초안)  
**작성일:** 2026-05-25  
**엔진:** Godot 4  
**레퍼런스:** Punch Club (Lazy Bear Games, 2016)  
**세계관:** MMA / UFC

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 격투가 육성 시뮬레이션 RPG |
| 플랫폼 | PC (Windows) |
| 플레이 시간 | 약 2~3시간 (엔딩 1회 기준) |
| 핵심 판타지 | 길거리 싸움꾼이 UFC 챔피언이 된다 |

플레이어는 3가지 격투 스타일 중 하나를 선택하고, 일과 관리(훈련/휴식/아르바이트)를 통해 스탯을 키우며 4개 티어를 순서대로 정복한다. 각 티어에는 플레이어 직업과 **상성상 불리한 라이벌**이 기다린다.

---

## 2. 직업 시스템 (Class System)

### 삼각 상성
```
      복서 (Boxer)
     /            \
  이김            짐
   /                \
주짓떼로 ——이김——> 레슬러
```

| 직업 | 이기는 상대 | 지는 상대 | 판타지 |
|------|------------|----------|--------|
| 복서 (Boxer) | 주짓떼로 | 레슬러 | 빠른 타격으로 거리 유지 |
| 레슬러 (Wrestler) | 복서 | 주짓떼로 | 태클로 타격가를 억누름 |
| 주짓떼로 (Jiu-Jitsu) | 레슬러 | 복서 | 그래플링으로 레슬러를 역이용 |

### 상성 배율
- 유리 매치업: 전투 최종 데미지 **+20%**
- 불리 매치업: 전투 최종 데미지 **-20%**
- 동일 직업: 배율 없음 (순수 스탯 싸움)

### 직업별 특화 스탯

| 직업 | 주력 스탯 | 고유 메카닉 |
|------|----------|-----------|
| 복서 | 민첩(AGI), 스트라이킹 | 콤보 카운터 — 연속 공격 시 데미지 누적 상승 |
| 레슬러 | 힘(STR), HP | 테이크다운 — 일정 확률로 상대를 넘어뜨려 추가 타격 |
| 주짓떼로 | 기술(TEC), 스태미나 | 서브미션 게이지 — 누적 시 즉사에 가까운 피니시 |

## 3. 콘텐츠 진행 구조 (4 티어)

```
[티어 1] 길거리 파이터 (Street Fighter)
    → 일반 대전 3명 → 라이벌 등장 → 구역 보스

[티어 2] KFC 파이터 (Korea Fighting Championship)
    → 일반 대전 4명 → 라이벌 재등장(강화) → KFC 타이틀 매치

[티어 3] UFC 파이터
    → 일반 대전 5명 → 라이벌 3차 등장 → UFC 타이틀 도전자 결정전

[티어 4] UFC 챔피언
    → 챔피언 방어전 2회 → 최종 라이벌 (최종 보스) → 엔딩
```

### 티어별 분위기

| 티어 | 배경 | 적 수준 | 보상 |
|------|------|--------|------|
| 1. 길거리 | 골목/공터 | 낮음, 무식한 싸움 | 소액 + 도장 입문 자격 |
| 2. KFC | 소규모 링 | 중간, 스타일 있음 | 계약금 + 스폰서 장비 |
| 3. UFC | 정식 옥타곤 | 높음, 전문 파이터 | 높은 파이트머니 |
| 4. 챔피언 | 대형 이벤트 | 최고, 라이벌 포함 | 벨트 + 엔딩 영상 |

## 4. 라이벌 시스템 (Rival System)

### 라이벌 배정 규칙
플레이어가 직업을 선택하면 **상성상 불리한 직업**의 라이벌이 자동 배정된다.

| 플레이어 직업 | 라이벌 직업 | 라이벌 이름 (예시) |
|--------------|-----------|-----------------|
| 복서 | 레슬러 | "마운틴" 김철수 |
| 레슬러 | 주짓떼로 | "스네이크" 박지훈 |
| 주짓떼로 | 복서 | "플래시" 이민준 |

### 라이벌 등장 패턴

| 등장 시점 | 상태 | 역할 |
|----------|------|------|
| 티어 1 클리어 직전 | 플레이어보다 강함 (패배 이벤트) | 동기부여 — "언젠가 꼭 이긴다" |
| 티어 2 보스 | 비슷한 수준 | 플레이어 성장 확인 |
| 티어 3 도전자 결정전 | 플레이어와 박빙 | 극적인 승부 |
| 티어 4 최종 보스 | 최강 버전 | 클라이맥스 |

### 라이벌 성장 설계 의도
- 라이벌도 티어에 따라 스탯이 올라가므로 플레이어는 항상 상성 불이익을 극복할 **스탯 차이**를 만들어야 한다
- 상성 -20% 디버프를 뒤집으려면 스탯 총합 약 **25% 이상** 우위 필요 (밸런스 시트 참조)

## 5. 스탯 시스템

### 5가지 기본 스탯

| 스탯 | 기호 | 영향 범위 | 훈련 방법 |
|------|------|----------|---------|
| 힘 (Strength) | STR | 기본 데미지 | 웨이트 트레이닝 |
| 민첩 (Agility) | AGI | 회피율, 공격 속도 | 달리기, 줄넘기 |
| 스태미나 (Stamina) | STA | 전투 지속 능력, 스킬 사용 | 사이클, 수영 |
| 기술 (Technique) | TEC | 직업 스킬 계수, 크리티컬 | 스파링, 미트 훈련 |
| HP (체력) | HP | 전투 생존력 | 식단 관리 |

### 스탯 감소 시스템 (Punch Club 핵심 메카닉 계승)

> "훈련하지 않으면 녹슨다"

- STR, AGI, TEC는 **매 게임 내 하루**가 지날 때마다 소폭 감소
- STA가 부족한 상태가 지속되면 감소 속도 **2배**
- HP는 자연 감소 없음 (식단으로만 올림)

```
기본 감소율 예시:
STR: -0.5/day  AGI: -0.7/day  TEC: -0.3/day
STA 부족 시: 위 수치 ×2
```

## 6. 훈련 시스템

### 하루 시간 슬롯 구조

하루는 **3개 슬롯** (아침 / 오후 / 저녁)으로 구성. 각 슬롯에 1가지 행동 배치.

| 행동 | 효과 | 비용 |
|------|------|------|
| 웨이트 트레이닝 | STR +X, STA -Y | 도장 멤버십 필요 |
| 달리기 | AGI +X, STA +X | 무료 |
| 스파링 | TEC +X, STA -Y, HP -Z | 도장 멤버십 필요 |
| 식단 관리 | HP +X, STA 회복 | 골드 소모 |
| 아르바이트 | 골드 획득 | 스탯 향상 없음 |
| 휴식 | STA 대폭 회복 | 무료 |

### 훈련 효율 곡선
- 스탯이 낮을수록 훈련 효율 높음 (초반 빠른 성장)
- 일정 수치 이상은 수확 체감 (후반 고효율 훈련 장비 필요)

## 7. 전투 시스템

### 전투 방식
Punch Club과 동일한 **자동 진행** 방식. 플레이어는 전투 전 **전략** 만 선택.

### 전투 전 전략 선택

| 전략 | 효과 |
|------|------|
| 공격적 (Aggressive) | 데미지 +15%, 방어 -15% |
| 균형 (Balanced) | 기본 스탯 그대로 |
| 방어적 (Defensive) | 방어 +15%, 데미지 -15% |

### 전투 데미지 공식 (초안)
```
기본 데미지 = (STR × 1.5 + TEC × 0.5) × 직업 스킬 계수
최종 데미지 = 기본 데미지 × 상성 배율 × 전략 배율 × 크리티컬 여부
회피 판정  = 상대 AGI ÷ (내 AGI + 상대 AGI)
```

### 전투 흐름 (1라운드 = 5분 게임 내 시간)
```
라운드 시작
→ 양측 공격 교환 (AGI 기반 공격 순서)
→ 특수 메카닉 발동 체크 (콤보/테이크다운/서브미션)
→ HP 0 또는 스태미나 0 → KO/서브미션 패배
→ 3라운드 종료 시 판정 (총 데미지 비교)
```

## 8. 경제 시스템

### 수입 원천

| 수입 | 금액 범위 | 조건 |
|------|----------|------|
| 아르바이트 | 50~100 골드/슬롯 | 슬롯 사용 |
| 시합 승리 보너스 | 200~2000 골드 | 티어 비례 |
| 스폰서 계약 | 500 골드 (1회) | 티어 2 진입 시 |

### 지출 항목

| 지출 | 금액 | 주기 |
|------|------|------|
| 도장 멤버십 | 300 골드 | 주간 |
| 식단 (영양제) | 50~150 골드 | 구매 시 |
| 장비 업그레이드 | 500~2000 골드 | 1회성 |

### 경제 루프 설계 의도
- 초반: 아르바이트와 훈련 사이 **시간 트레이드오프**가 핵심 긴장감
- 중반: 시합 보너스로 아르바이트 의존도 감소, 훈련 집중 가능
- 후반: 돈보다 **시간 효율 극대화**가 더 중요한 자원

## 9. 스킬 트리 (개요)

각 직업당 **9개 스킬** (3×3 트리 구조). 스킬 포인트는 시합 승리 시 획득.

### 복서 스킬 트리 (예시)
```
[잽 마스터] → [원투 콤보] → [스웨이 백]
      ↓               ↓              ↓
[풋워크]    → [훅 파워업]  → [더킹]
      ↓               ↓              ↓
[바디샷]    → [어퍼컷]    → [KO 피니셔]
```
