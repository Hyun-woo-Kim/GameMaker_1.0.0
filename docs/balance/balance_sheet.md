## 3. 훈련 효율표

| 훈련 | 상승 스탯 | 기본 +값 | 소모 STA | 비용 |
|------|---------|---------|---------|------|
| 웨이트 | STR | +3 | -5 | 멤버십 |
| 달리기 | AGI | +2 | +1 | 무료 |
| 줄넘기 | AGI | +1, STA +2 | 0 | 무료 |
| 스파링 | TEC | +3 | -8 | 멤버십 |
| 식단 | HP | +10 | STA +5 | 50골드 |
| 휴식 | — | — | STA +15 | 무료 |

**수확 체감 공식:**
```
실제 +값 = 기본 +값 × (목표 스탯 ÷ 현재 스탯) ^ 0.4
예) 현재 STR=40, 목표=80: 3 × (80÷40)^0.4 = 3 × 1.32 = +3.96 → 약 +4
    현재 STR=75, 목표=80: 3 × (80÷75)^0.4 = 3 × 1.03 = +3.09 → 약 +3
```

## 4. 전투 데미지 공식 (상세)

### 기본 공격력
```
ATK = (STR × 1.5 + TEC × 0.5) × 직업 계수
```

| 직업 | 직업 계수 | 특성 |
|------|---------|------|
| 복서 | 1.0 (ATK 기준) | 빠른 공격 (AGI 보너스 추가) |
| 레슬러 | 1.1 (STR 비중 높음) | 느리지만 강함 |
| 주짓떼로 | 0.9 (TEC 비중 높음) | 서브미션 게이지로 보완 |

### 최종 데미지
```
최종 데미지 = ATK × 상성 배율 × 전략 배율 × 크리티컬 배율
```

| 변수 | 유리 | 중립 | 불리 |
|------|------|------|------|
| 상성 배율 | 1.20 | 1.00 | 0.80 |

| 전략 | 공격 배율 | 방어 배율 |
|------|---------|---------|
| 공격적 | 1.15 | 0.85 |
| 균형 | 1.00 | 1.00 |
| 방어적 | 0.85 | 1.15 |

| 크리티컬 | 배율 | 발동 확률 |
|---------|------|---------|
| 크리티컬 발동 | 1.5 | TEC ÷ 200 |
| 미발동 | 1.0 | — |

### 회피 판정
```
회피율 = 상대 AGI ÷ (내 AGI + 상대 AGI) × 조정 계수(0.6)
최대 회피율: 40% 캡
```

## 5. 상성 극복 조건 (핵심 밸런스)

라이벌은 플레이어에게 상성 **불리** 직업. 상성 -20%를 뒤집으려면 순수 스탯으로 보완해야 함.

**스탯 총합 기준:**
```
동일 스탯 대결 시 불리 측의 기대 승률: ~38%
상성 극복 목표 승률: 50% 이상
필요 스탯 우위: 약 +27% (1.20÷0.80 = 1.5배 차이를 스탯으로 상쇄)
```

| 티어 | 라이벌 스탯 총합 | 플레이어 목표 스탯 총합 | 필요 우위 |
|------|--------------|-------------------|---------|
| 1 | 80 | 90 | +12.5% |
| 2 | 160 | 180 | +12.5% |
| 3 | 250 | 285 | +14% |
| 4 | 340 | 390 | +14.7% |

> 라이벌 티어가 올라갈수록 요구 우위가 소폭 증가 → 후반부 난이도 자연 상승

## 6. 전투 지속 시간 설계

목표: 1라운드 = 실제 30~45초. 3라운드 제도.

```
라운드당 예상 타격 횟수 = 5회 (각 0.5데미지 × STR 기준)
평균 HP 소모율: 라운드당 HP의 20~25%
→ 3라운드 정상 진행 시 HP 60~75% 소모
→ KO 발생 확률: 스탯 차이 20% 이상일 때 ~65%
```

## 7. 경제 밸런스

### 초반 (티어 1) 예상 현금 흐름

```
주간 지출:
  도장 멤버십: 300골드
  식단: 50골드 × 7 = 350골드
  총 지출: 650골드/주

주간 수입 (아르바이트 50%):
  3슬롯/day × 7day × 50% × 75골드 = ~787골드/주

순이익: +137골드/주 (아르바이트 비중이 높을수록 훈련 부족)
```

**의도:** 초반에는 아르바이트와 훈련 사이에서 항상 부족함을 느끼게 설계.  
시합 승리 보너스(200골드)가 큰 동기 부여가 됨.
